Inflector 輔助函數¶

Inflector 輔助函數檔案包含允許你更改 英語 為複數、單數、駝峰式諸類型式的功能

  • 載入此輔助函數

  • 可用的功能

載入此輔助函數¶

此輔助函數可以利用以下的程式碼載入：

helper('inflector');

可用的功能¶

以下敘述的功能皆可用：

singular($string)¶
Parameters

  • $string (string) – 輸入字串

Returns

一個單數詞

Return type

string

將複數詞變更為單數詞。例如:

echo singular('dogs'); // 印出 'dog'
plural($string)¶
Parameters

  • $string (string) – 輸入字串

Returns

一個複數字

Return type

string

將一個單數詞變更為複數詞。例如:

echo plural('dog'); // 印出 'dogs'
counted($count, $string)¶
Parameters

  • $count (int) – 物品的數字

  • $string (string) – 輸入數字

Returns

一個單數或複數詞語

Return type

string

將一個字詞與它的數量變更為一個詞語。例如:

echo counted(3, 'dog'); // 印出 '3 dogs'
camelize($string)¶
Parameters

  • $string (string) – 輸入字串Input string

Returns

駝峰式字串

Return type

string

將一個字串中以空格隔開或是畫底線的詞語合併成為駝峰式。例如:

echo camelize('my_dog_spot'); // 印出 'myDogSpot'
pascalize($string)¶
Parameters

  • $string (string) – 輸入字串

Returns

Pascal 字串

Return type

string

將一個字串中以空格隔開或是畫底線的詞語合併成 Pascal 型式，意即首字為大寫的駝峰式命名。例如:

echo pascalize('my_dog_spot'); // 印出 'MyDogSpot'
underscore($string)¶
Parameters

  • $string (string) – 輸入字串

Returns

字串包含底線而非空格

Return type

string

選取多個空格隔開的字並改為底線。 例如:

echo underscore('my dog spot'); // 印出 'my_dog_spot'
humanize($string[, $separator = '_'])¶
Parameters

  • $string (string) – 輸入字串

  • $separator (string) – 輸入分隔符號

Returns

人性化的字串

Return type

string

選取多個由底線隔開的字並在其中增加空格。每一個詞的第一個字都是大寫的。

例如:

echo humanize('my_dog_spot'); // 印出 'My Dog Spot'

使用破折號而非底線:

echo humanize('my-dog-spot', '-'); // 印出 'My Dog Spot'
is_pluralizable($word)¶
Parameters

  • $word (string) – 輸入字串

Returns

TRUE：當字詞是可數的 或 FALSE：當字詞不可數

Return type

bool

確認是否給定的字詞有複數形式。例如:

is_pluralizable('equipment'); // 回傳 FALSE
dasherize($string)¶
Parameters

  • $string (string) – 輸入字串

Returns

使用破折號的字串

Return type

string

在字串中以破折號取代底線。例如:

dasherize('hello_world'); // 回傳 'hello-world'
ordinal($integer)¶
Parameters

  • $integer (int) – 定義後綴的數字

Returns

序號的後綴

Return type

string

回傳應該加在數字後面以表示序位的後綴，像是 1st, 2nd, 3rd, 4th。例如:

ordinal(1); // 回傳 'st'
ordinalize($integer)¶
Parameters

  • $integer (int) – 要排序的整數

Returns

排序過的整數

Return type

string

將一個數字字串列轉換為表示序位的序數字串，像是 1st, 2nd, 3rd, 4th。 例如:

ordinalize(1); // 回傳 '1st'
